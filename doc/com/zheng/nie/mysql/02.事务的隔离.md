#####事务四大特性
    1.A
        Atomicity:原子性
    2.C
        Consistency：一致性
    3.I
        Isolation：隔离性
    4.D
        Durability：持久性
#####视图
    1.在可重复读的隔离级别下视图在事务启动的时候进行创建，
    2.在读提交的隔离级别下这个视图是在每个sql执行的时候创建的。
    3.在读未提交的隔离级别下没有视图概念
#####事务隔离级别的概念
#####事务的启动方式
    1.显示启动
        begin 或 start transaction。配套的提交语句是 commit，回滚语句是 rollback。
    2.set autocommit=0
        这个命令会将这个线程的自动提交关掉，必须显示的提交事务，有可能会出现长事务的情况。
    3.set autocommit=1
        必选显示的启动事务
